# 助眠苹果手机版 — 设计文档

**日期**：2026-04-11
**状态**：已批准

## 概述

一款面向 iPhone 用户的 PWA 助眠应用，通过三阶段音频会话（引导→深睡→唤醒）+ 双耳节拍 + 场景化动画，帮助用户入睡和唤醒。

## 技术方案

**方案**：纯前端 PWA（方案 A）

- 无后端依赖，完全离线可用
- 后续可扩展云同步和内置音频库

## 技术栈

| 领域 | 技术 |
|------|------|
| 音频处理 | Web Audio API（AudioContext, GainNode, OscillatorNode, StereoPannerNode） |
| 动画渲染 | Canvas 2D + CSS 动画 |
| 数据存储 | localStorage（设置）+ IndexedDB（音乐文件） |
| 离线缓存 | Service Worker |
| 前端框架 | 纯原生 HTML/CSS/JS，无构建工具 |

## 三阶段睡眠会话引擎

### 时间线模型

```
|<────── 总时长（用户设定，如 30 分钟）──────────────>|
|── 引导阶段 ──|────── 深睡阶段 ─────|── 唤醒阶段 ──|
    15%              70%                15%
```

- 三个阶段按比例自动划分（15% / 70% / 15%），用户只需设置总时长
- 高级设置中可自定义比例

### 各阶段行为

| 阶段 | 音乐音量 | 双耳节拍频率 | 时长占比 |
|------|---------|------------|---------|
| 引导阶段 | 80%→50%（渐降） | Theta（4-8Hz）→ Delta（0.5-4Hz） | 15% |
| 深睡阶段 | 50%→20%（缓慢渐降） | Delta（0.5-2Hz） | 70% |
| 唤醒阶段 | 20%→80%（渐升） | Delta→Alpha（8-12Hz） | 15% |

### 渐变实现

使用 `AudioParam.linearRampToValueAtTime()` 实现音量和频率的平滑过渡，避免突兀跳变。

### 用户可调参数

- 总时长（滑块：10-90 分钟）
- 三阶段比例（高级设置）
- 各阶段目标音量范围
- 双耳节拍基础频率 + 节拍频率

## 音频引擎

### 信号流

```
用户音乐文件 ──→ MediaElementSource ──→ GainNode（音量控制）──┐
                                                              ├──→ Destination（扬声器）
双耳节拍生成器 ──→ OscillatorNode(左) + OscillatorNode(右) ──┘
```

### 双耳节拍实现

- **原理**：左耳播放 `baseFreq`，右耳播放 `baseFreq + beatFreq`，大脑感知差值频率
- **实现**：两个 OscillatorNode 分别连接 StereoPannerNode（左 -1 / 右 +1）
- **用户可调**：基础频率（100-500Hz）、节拍频率（0.5-40Hz）
- **波形**：正弦波（最柔和）

### 音乐文件处理

- 支持格式：MP3、AAC、OGG、WAV
- `<input type="file">` 选择，`URL.createObjectURL()` 播放
- IndexedDB 持久化存储

## 场景动画系统

### 1. 星空夜景

- **背景**：深蓝到黑色渐变
- **元素**：闪烁星星（不同大小亮度）、流星、月亮
- **氛围**：宁静深邃

### 2. 海滩海浪

- **背景**：深蓝夜空 + 深色海面
- **元素**：多层 CSS 波浪动画、月光水面倒影、远处灯塔微光
- **氛围**：温柔有节奏

### 3. 森林自然

- **背景**：深绿色渐变
- **元素**：树木剪影、萤火虫（发光粒子）、树叶飘落
- **氛围**：神秘安宁

### 4. 花田草地

- **背景**：深紫/蓝色夜空 + 草地
- **元素**：花瓣飘落、草地微风摆动、远处山丘剪影
- **氛围**：柔和浪漫

### 技术实现

- 每个 Scene 独立渲染类，继承 BaseScene
- `requestAnimationFrame` 驱动动画循环
- 场景切换淡入淡出过渡
- 省电模式：CSS filter 自适应降低亮度

## 界面布局

### 主界面

- 全屏场景动画背景
- 顶部：场景选择器（四缩略图横向滑动）
- 中部：时长滑块 + 音乐文件选择
- 底部：双耳节拍设置面板（可折叠）+ 开始按钮

### 运行中界面

- 全屏场景动画背景
- 中央：倒计时 + 进度条 + 当前阶段指示
- 底部：暂停/停止按钮
- 底部上滑展开高级设置

## 边界处理

- 无音乐文件时：仅播放双耳节拍并提示用户
- 来电/闹钟中断：监听 pagevisibility 暂停会话
- 浏览器不支持 Web Audio API：友好提示
- PWA 安装引导：首次访问提示"添加到主屏幕"
- Wake Lock API：锁屏后音频持续播放

## 离线支持

- Service Worker 缓存所有静态资源
- 用户音乐文件存储在 IndexedDB
- 完全断网可正常使用
